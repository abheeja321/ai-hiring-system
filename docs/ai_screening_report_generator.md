# AI Screening Report Generator

## Report sections

- summary
- key answers
- strengths
- risks
- missing data
- recommendation
- per-question scores

## Recruiter-ready highlights

- salary expectation
- availability
- confirmed skills
- strong or weak communication signals
- off-topic, vague, or contradictory answers

## Exportable report format

The report is emitted as structured JSON and can be rendered into HTML, PDF, or recruiter dashboard cards.
